Trivial: Símbolos Químicos

Trivial de química para aprender los símbolos químicos de los elementos de la tabla periódica. Apto para todas las edades, estudiantes de secundaria, bachillerato y todo aquellos interesados en aprender o recordar los símbolos químicos.

Técnologías utilizadas:

  • HTML5
  • CC3
  • JavaScript
  • jQuery Mobile
  • LocalStorage para almacenar datos estadísticos
  • PhoneGap
  • Responsive Web Design
